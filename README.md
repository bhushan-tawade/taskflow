# TaskFlow -- Task Management Web App

TaskFlow is a simple Task Management Web Application that helps users
organize and track their daily tasks efficiently.\
Users can create tasks, manage priorities, mark them as completed, and
track productivity in a clean dashboard.

------------------------------------------------------------------------

## 🚀 Features

-   Add a new task with title, description, category, priority, and due
    date
-   View all tasks in a dashboard
-   Mark tasks as completed or pending
-   Delete tasks
-   Filter tasks (All, Pending, Completed, Overdue)
-   Search tasks
-   Task statistics cards
-   Drag & Drop Kanban board
-   Dark mode toggle
-   Toast notifications

------------------------------------------------------------------------

## 🛠 Tech Stack

### Frontend

-   React.js

-   React Toastify
-   React Icons

### Backend

-   Node.js

-   MongoDB

## ⚙️ Installation & Setup

### 1. Clone the Repository

    git clone https://github.com/your-username/taskflow.git
    cd taskflow

### 2. Install Backend Dependencies

    cd backend
    npm install
    npm start

### 3. Install Frontend Dependencies

    cd frontend
    npm install
    npm run dev
